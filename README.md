# Terraform Lab 3
Project demonstrates how to define, deploy, and destroy resources using Terraform.

---

## Prerequisites

Before you begin, ensure you have the following installed and configured:

- [Terraform](https://developer.hashicorp.com/terraform/downloads) (version 1.x or later)
- [AWS CLI](https://docs.aws.amazon.com/cli/) (if using AWS)
- Configured AWS credentials:
- An active AWS account with appropriate permissions

---

## Usage Instructions

### 1. Initialize Terraform
```
terraform init
```

---

### 2. Apply Configuration
```
terraform apply
```
---

### 3. Destroy Infrastructure
```
terraform destroy
```
